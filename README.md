# Business Website

A simple, professional website for showcasing your business work and materials.

## Getting Started

1. **Open the website**: Simply double-click on `index.html` to open it in your browser

2. **Access Admin Panel**: 
   - Open `admin.html` in your browser
   - Default password: **admin123**
   - Use the admin panel to preview and organize your images

3. **Add Your Images**: 
   - Use the admin panel to select images (it will guide you)
   - OR manually place your work photos in the `images/` folder with these names:
     - work1.jpg, work2.jpg, work3.jpg, work4.jpg, work5.jpg, work6.jpg
   - Place your raw materials photos with these names:
     - material1.jpg, material2.jpg, material3.jpg, material4.jpg
   - The website will use placeholder images until you add your own

4. **Customize Content**:
   - Open `index.html` in a text editor
   - Replace "My Business" with your business name
   - Update the contact information (address, email, phone, hours)
   - Modify project descriptions and captions

## Admin Panel Features

🔐 **Password Protected** - Only you can access it  
📸 **Image Upload Interface** - Easy drag-and-drop or click to browse  
🛠️ **Manage Content** - Update business info and contact details  
⚙️ **Settings** - Change your admin password  
📊 **Dashboard** - See what's uploaded at a glance

## Tips

- Image format: Use .jpg, .jpeg, or .png files
- For best results, use images with similar dimensions
- All images will be automatically resized to fit
